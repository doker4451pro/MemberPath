# MemberPath
Когда нужно проинизиализировать кучу полей у какого-то класса в разных объектах. Можно использовать рефлексию, но прописывать названия долго и можно ошибиться лучше сделать возможность для каждого объекта на сцене выбрать из выпадающего меню нужное поле для инициализации.
